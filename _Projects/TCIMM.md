## 簡介
TCIMM (Traditional Chinese Input Method Material) 致力於繁體中文輸入法素材收集。項目包括選字、字的構成、詞、短語、名句等輸入法的構成部件。  
項目最初放在 Google Code ，後轉 Google Sites 。現更名並放在 GitHub 。
